Doctoral Database and Query Website

Created by Juan Diego Gonzalez German (2019)

This site allows you to connect to the Doctoral UTA Database.
Operations included in V1.0:

-Searching for Students and Faculty
    This can be done by either entering their first and last names, or their unique ID

-Adding Students to the DB
    Only Self Supported Students implemented for now
    All their information is needed. Their 6 characters ID needs to be chosen manually

-Removing Students to the DB
    Needs to be done with their unique ID
    To find a student's ID, search the DB with their full name

-Updating Faculty
    Faculty member type can be changed via a radio button
    This needs to be done with their unique ID
    To find a faculty member's ID, search the DB with their full name
